miur
####

**MIUR = Modular Interactive Universal exploreR**

MIUR is an experimental *graph-based* explorer that lets a user **navigate, preview, and operate**
on heterogeneous “things” (filesystem paths, text blobs, process outputs, workflows, overlays/annotations,
web URLs, …) through one uniform protocol.
